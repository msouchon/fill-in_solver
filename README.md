# fill-in_solver
Solves [Fill-In](https://en.wikipedia.org/wiki/Fill-In_(puzzle)) Puzzles

Given a blank puzzle words and a list of words this program outputs a solved board file.

For example this is puzzle 2:
Board:
```
____
___#
____
```

Words:
```
boat
art
need
ban
ore
ate
```

Solution:
```
boat
art#
need
```
